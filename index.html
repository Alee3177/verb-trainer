2. 動詞規則總覽

動詞三大類

類型 日文 平假名 / 羅馬拼音 中文
五段動詞書く / 読む / 話すかく (kaku) / よむ (yomu) / はなす (hanasu)寫 / 讀 / 說
一段動詞食べる / 見る / 寝るたべる (taberu) / みる (miru) / ねる (neru)吃 / 看 / 睡
不規則動詞する / 来るする (suru) / くる (kuru)做 / 來

五段動詞口訣

口訣 規則 例子
一起立う／つ／る → って会う → 会って、待つ → 待って、取る → 取って
你咪咪ぶ／む／ぬ → んで遊ぶ → 遊んで、読む → 読んで、死ぬ → 死んで
く行く → いて書く → 書いて、聞く → 聞いて
ぐ行ぐ → いで泳ぐ → 泳いで、急ぐ → 急いで
す行す → して話す → 話して、貸す → 貸して
例外行く → 行って行く → 行って
變化類型 規則 例子
て形連接、請求、順接書いて、食べて、来て
ない形否定書かない、食べない、来ない
た形過去書いた、食べた、来た

4. 快速記憶卡

五段動詞

尾音會變
最常見
要先分類

例：書く、読む、話す、待つ、取る、帰る

一段動詞

多為 る 結尾
多半去る
較規則

例：食べる、見る、起きる、寝る

不規則動詞

數量少
直接記

例：する、来る

學習順序

分類
て形
ない形
た形

先判斷，再變形，效率最高。

5. 學習路線

階段 重點 建議做法
第 1 階段動詞分類每天 20 題，先把五段 / 一段 / 不規則判斷熟
第 2 階段て形從五段規則切入，練出反射
第 3 階段ない形 / た形用同一批動詞反覆練，建立系統感
第 4 階段口說結合開自動播放，跟著念，做到看到就能說
